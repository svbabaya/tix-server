## How to setup environment for development tixerver and tixtune

Руководство по установке и настройке среды разработки **tixerver** и **tixtune** состоит из нескольких инструкций.

**tixerver** - приложение для установки на камеру
**tixtune** - приложение для отладки алгоритма, которое работает с тем же алгоритмическим модулем, что и tixerver. И если tixerver собирается с применением кросскомпиляторов для соответствующей камеры, то алгоритмический модуль для tixtune собирается под Linux с помощью gcc/g++. Остальные модули tixtune планируется делать на Python/Qt
