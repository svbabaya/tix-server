## How to install WSL on Windows

WSL (Subsystem for Linux) — это технология от Microsoft, которая позволяет запускать полноценную среду Linux прямо в Windows, без необходимости устанавливать виртуальную машину или двойную загрузку системы. Внутри работает легковесная виртуальная машина с реальным ядром Linux, а значит можно использовать терминал Linux (bash, zsh) и инструменты (git, ssh, grep, awk) прямо в Windows.
